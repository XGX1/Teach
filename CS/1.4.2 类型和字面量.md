# 内容
## 类型(type)
定义：能够容纳什么样子的数据，如杯子可以容纳水，但是不能容纳货物和大象
左侧的int就被称作类型，之后会讲解其他类型
int类型专门用来存储整数
```c#
int value;
```
## 字面量
定义：直接写在代码中的值
以下代码中的1就是字面量
```cs
int value = 1; 
```
字面量编译后无法被修改，之后会讲如何如何编译后修改值 ^e97a58
