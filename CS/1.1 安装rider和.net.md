## 新建项目
### dotnet方式 
新建项目
```fish
dotnet new console -n ConsoleLearnTemp
```
进入项目目录
```fish
cd ConsoleLearnTemp
```
运行项目,注意运行的是当前目录的程序，请确保自己进入了项目目录
```fish
dotnet run
```

第一句是解释编辑程序所在的位置
第二句是输出的内容
第三句告诉你运行已经结束了，返回的代码是多少，0就是代码返回正确
可以通过修改代码中""之间的内容，改变输出的内容
若是显示内容不同，尤其是出现红色文字的情况，请重新安装.net，.net可能安装没有成功
