#### 字体介绍
noto sans mono cjk sc：包含了大量的字体，并且是等宽字体，不仅是英文等宽，中文和英文也是等宽的，cjk sc 指简体中文。
Cascadia Code pl ：支持连字符比较好看，而且一些字符比如0和o比较容易区分
